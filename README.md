# ⚔️ Battle of Moves

Battle of Moves is a fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🎮 Features

* Rock, Paper, Scissors gameplay
* Score tracking
* Smooth animations
* Responsive design
* Interactive hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Play

1. Choose Rock, Paper, or Scissors.
2. The computer makes its move.
3. The winner is displayed instantly.
4. Keep playing and beat the computer!
